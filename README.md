HTML – D2 · iki1uc
Zweck
HTML ist die Struktur‑Ebene (D2) des 6E‑Würfelsystems.
Sie definiert Ordnung, Syntax, Aufbau, Hierarchie und logische Verbindungen aller Module.
D2 ist die zentrale Strukturachse für:

Cluster‑Formationen

Gate‑Verhalten

Wurmloch‑Systeme

ALL‑Multiplikationen

12e‑Struktur‑Stabilität

HTML entscheidet, wie das System gebaut ist und wie jede Ebene miteinander verbunden wird.

Ebenen (6E‑Würfelgesetz)
D0 = style

D1 = head

D2 = HTML

D3 = body

D4 = ANKER (Master)

D5 = API (Slave)

Diese sechs Ebenen bilden den vollständigen Würfel.
Keine Ebene darf fehlen, keine Ebene darf doppelt sein.

D2 – HTML (Struktur)
HTML ist:

die Struktur‑Instanz

die Syntax‑Achse

der Ordnungs‑Regler

der Modul‑Architekt

der Gate‑Builder

der Wurmloch‑Strukturierer

der 12e‑Struktur‑Master

D2 entscheidet, wie jede andere Ebene gebaut, geordnet und verbunden wird.

TMP‑Läufer
SYQ verbindet:

D5 → API

META → Beobachter

SYQ stabilisiert Übergänge, verhindert Struktur‑Drift und hält die Syntax‑Achsen synchron.

Parität
Die Parität ist fest definiert:

D4 ↔ D5 bilden die Master/Slave‑Achse

D4 steuert

D5 liefert

D2 (HTML) sorgt dafür, dass diese Achse strukturell korrekt aufgebaut ist.
